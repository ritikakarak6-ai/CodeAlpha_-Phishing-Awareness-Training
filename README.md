# CodeAlpha Internship - Cyber Security

## TASK 2: Phishing Awareness Training

This repository contains the presentation module developed during my Cyber Security Internship at **CodeAlpha** for Phishing Awareness Training.

### 📌 Project Overview
Phishing and social engineering are among the most critical security threats facing organizations today. This training module is designed to educate users on how to identify fraudulent emails, detect spoofed websites, and adopt cybersecurity best practices to mitigate human-factor risks.

### ⚙️ Topics Covered
- **Introduction to Phishing & Social Engineering:** Understanding the psychology behind the scams.
- **Types of Phishing:** Email Phishing, Spear Phishing, Whaling, Smishing, and Vishing.
- **Red Flags Identification:** Recognizing urgent language, fake sender domains, and malicious attachments.
- **Spotting Fake Websites:** Analyzing URLs, checking for HTTPS, and spotting typosquatting[cite: 1].
- **Interactive Awareness Quiz:** Real-world scenarios to test and reinforce learning[cite: 1].
